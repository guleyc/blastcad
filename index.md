# Welcome to BlastCAD

**BlastCAD** is a modern, browser-based CAD tool designed for engineers, designers, and makers who need fast, precise 2D and 3D drafting without installing anything.

{: .note }
> BlastCAD is under active development. Features and APIs may change between releases. Check the [Changelog](changelog) for the latest updates.

---

## What is BlastCAD?

BlastCAD brings professional-grade CAD capabilities directly to your browser. Whether you are sketching a quick part concept or producing detailed technical drawings, BlastCAD has the tools you need — with zero setup.

**Key highlights:**

- 🚀 **Instant access** — No installation. Open a browser, start designing.
- 📐 **Precision drafting** — Snapping, constraints, and dimensioning built in.
- 🗂️ **Multiple file formats** — Import and export DXF, SVG, and more.
- ☁️ **Cloud-ready** — Save and share your designs from anywhere.
- 🔌 **Extensible** — Build on BlastCAD with the scripting API.
