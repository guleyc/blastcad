## [0.1.0] — Initial Release

### Added
- Browser-based CAD workspace — no installation required.
- **Drawing tools:** Line, Rectangle, Circle, Arc, Polyline, Text.
- **Modify tools:** Move, Rotate, Scale, Mirror, Offset, Trim, Extend.
- **Dimension tools:** Linear, Aligned, Radial, Diameter, Angular.
- Grid with configurable spacing and snap-to-grid.
- Object snapping (endpoint, midpoint, intersection, perpendicular, tangent).
- Layer management (create, rename, delete, show/hide, lock).
- **File formats:** native `.blastcad`, DXF import/export, SVG import/export, PDF export, PNG/JPEG export.
- Undo / Redo (unlimited history).
- Auto-save to browser local storage (every 30 seconds).
- Document templates: Blank, A4/Letter, ISO Metric.
- Keyboard shortcut system with in-app reference overlay (`?`).
- Properties panel for viewing and editing selected object attributes.
