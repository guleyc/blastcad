# Charging & Explosives Design

The Charging Module allows engineers to define complex explosive columns with real-time feedback on density and quantities.

### Key Features
* **3D Sectional View:** Visualize explosive segments within the hole using high-fidelity rendering.
* **Explosives Database:** Choose from predefined materials like Emulsion 1.0, ANFO, or custom stemming types.
* **Primer Placement:** Manually or automatically place primers/detonators at specific depths from the collar or toe.
* **Proximity Analysis:** Dynamic distance lines show the "Burden" between the cursor and adjacent holes in real-time.

### Calculations
BlastCAD calculates the **Quantity (kg)** of explosives based on:
1.  **Hole Diameter:** Specified in the drill pattern.
2.  **Segment Length:** Defined by start and end depths.
3.  **Material Density:** Pulled from the explosives database.
