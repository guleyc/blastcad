# Data Exchange & Interoperability

[cite_start]BlastCAD acts as a central hub between major mining software. 

### Supported Formats
* [cite_start]**Surpac (.str):** Full support for Y,X,Z string formats including String ID identification. 
* [cite_start]**Micromine (.str):** Import complex string variables directly into the 2D workspace. 
* [cite_start]**AutoCAD (.dxf):** Export your rings and holes as 3D entities for site surveyors. [cite: 7]
* [cite_start]**BlastCAD Excel (.xlsx):** A comprehensive backup format that preserves layers, charging rules, and global origins.
