# Modification & Editing

BlastCAD features a professional-grade geometric kernel for entity modification. [cite: 8]

| Command | Description | Shortcut |
| :--- | :--- | :--- |
| **Trim** | Removes segments of entities where they intersect with others.  | `TRIM` |
| **Extend** | Projects the endpoint of a line or polyline to a selected boundary. [cite: 8] | `EXTEND` |
| **Offset** | Creates a concentric copy of the selected object at a specified distance. [cite: 8] | `OFFSET` |
| **Join** | Merges multiple line/polyline segments into a single continuous entity.  | `JOIN` |

> **Pro Tip:** Use the **Command Terminal** for AutoCAD-like precision. You can type relative coordinates (e.g., `@10,5`) or distances directly.
